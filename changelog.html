Version 1.3.1

Mostly internal changes.

Version 1.3.0

Custom Albums with minor improvements, and rehaul of database code.
Changes:

  1. Basic UI improvements like set volume being stored between sessions, and shortcuts for loop and shuffle.
  2. Added blurred album as background for remote window.
  3. Added ability to add, modify and delete custom albums.

Bugs fixed:
  1. Initialially, shuffle may be shown as enbaled even though songs weren't shuffled.
  2. Closing the player contols didn't transition out.

Version 1.2.2

Mainly internal changes and bug fixes.
Changes:

  1. Frame was removed from remote window.
  2. Added help window

Bugs fixed:
  1. Even if there weren't any 'Top Song' or 'Top Albums', the header was shown.
  2. Remote window pointed to app icon in the frontend rather than local app icon.
  3. Everytime an individual artist/album is shown, the placeholder image caused an error(it wasnt a proper image). Now a default image is shown as placeholder.
  4. Number of songs in an album would go out of sync due to bad handling. Handling has been improved and checks for number of songs has been added.
  5. When directory was changed in settings, the db wouldnt be updated with songs in new directory
  6. Any non mp3 files in the music directory would be also be included in everything. Now they are ignored unless they have the .mp3 extension.
